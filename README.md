# Web Sphere Generator

[*Web Sphere Generator* page](https://blackwhiteyoshi.github.io/WebSphereGenerator/site.html)

This application renders a sphere with a compute shader using the Phong reflection model.

There are no third-party dependencies involved for this project, just using the browser functionalities.
The only third-party software used is Rspack for building the JavaScript.


## Description

You can create an image of a sphere and save it as an .png file.  
You can customize the size of the sphere, its color, the background color, light direction and the light Phong parameters.  
Last you can choose your resolution and create a .png file.
